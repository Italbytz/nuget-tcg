# nuget-tcg

`nuget-tcg` provides reusable trading card game contracts, provider clients, and in-memory adapters for .NET applications.

Use this package family if you want to model cards, sets, and external catalog lookups without coupling your application to a single game or transport API.

## Packages

- `Italbytz.Tcg.Abstractions`
- `Italbytz.Tcg.Tcgdex`
- `Italbytz.Tcg.Scryfall`
- `Italbytz.Tcg.InMemory`

## What you can do with nuget-tcg

- define game-agnostic card and set models through shared abstractions
- call TCGDex or Scryfall through dedicated package clients
- use in-memory repositories for tests, prototypes, and local caching
